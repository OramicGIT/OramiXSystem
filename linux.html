Особенности

OramiX System использует рабочий стол XFCE4:

  • Лёгкий и быстрый интерфейс, идеально для старых и новых ПК.
  • Полностью настраиваемый рабочий стол.
  • Минимальное потребление ресурсов при максимальной функциональности.
  • Поддержка тем, панелей и виджетов для комфортной работы.

Программы и драйверы

OramiX System поставляется с основным набором программ:

  • Базовые утилиты для работы: текстовые редакторы, файловый менеджер, браузер.
  • Программы для программирования: компиляторы, IDE, терминалы.
  • Игры и развлекательный софт.
  • Драйвер NVIDIA для видеокарт.
  • Wine для запуска Windows-приложений.
  • Neofetch и другие утилиты для мониторинга системы.
